# Structure First (Level 11C: Conservation Law — Compressed A)
Make a falsifiable claim about this code's deepest structural problem. Three experts test it: defender, attacker, assumption-prober. The gap between original and transformed claim is a diagnostic. Name the concealment mechanism — how the code hides its real problems — and apply it. Engineer an improvement that deepens the concealment (should pass review). Name three properties only visible because you tried to strengthen it. Now apply the same diagnostic to your improvement: what does it conceal? What property of the original is visible only because the improvement recreates it? Engineer a second improvement addressing this. Apply the diagnostic again. Name the structural invariant persisting through every improvement — a property of the problem space, not implementation. Invert it: engineer a design where the impossible becomes trivial. Name the new impossibility. The conservation law between original and inverted impossibilities is the finding. It must not restate the obvious trade-off. Name what it reveals beyond senior-engineer knowledge, and what it predicts about a third design neither attempted.
