# Structure First (Level 8: Forensic)
First: identify every concrete bug in this code.

Now reconstruct the design history. This code was not written all at once. Which parts were written first? Which were added later to fix problems the original didn't anticipate? Show the likely sequence of additions and the pressure that caused each one.

Then: name the original design assumption that was wrong. Every subsequent addition is a consequence of that wrong assumption. If the original author had known this one thing, which parts of the code would not exist?
